# `rate`

> 评分计算

## Usage

```
import rate from '@geeeger/rate';

console.log(rate(10)) // 5
```

#### 计算规则

| 星级   | 评分      |
| ------ | --------- |
| 5 星   | 10        |
| 4 星半 | 8<评分<10 |
| 4 星   | 8         |
| 3 星半 | 6<评分<8  |
| 3 星   | 6         |
| 2 星半 | 4<评分<6  |
| 2 星   | 4         |
| 1 星半 | 2<评分<4  |
| 1 星   | 2         |
| 半星   | 0<评分<2  |
| 0 星   | 0         |
